# CWGFR
该项目为LZU人工智能实践配套工作的图像预处理部分。主要的数据集是自OpenI数据集[6]的图像和报告。

由SSR->MSR->GIF->GIFR->WGIF->WGIFR->CWGIF->CWGIFR的顺序实现CWGFR复现[5]，但效果不如原文，尚需精调超参数。

./img内为提供的一些图像以供Notebook运行，请修改所有读取路径以便正确使用图片。
